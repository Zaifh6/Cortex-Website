# React + Vite

This project is set up using Vite with React, providing a fast and efficient development experience with hot module replacement (HMR) out of the box.

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
```

## Features

* Fast development with Vite
* Hot Module Replacement (HMR)
* Clean and minimal project structure
* ESLint configured for better code quality

## Notes

This setup is intentionally kept simple and flexible so you can scale it according to your project needs. You can easily extend it by adding TypeScript, additional linting rules, or other tools as your application grows.
